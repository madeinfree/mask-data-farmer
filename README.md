# Mask Data Farmer

Taiwan mask data proxy by GraphQL

# Demo

[健保特約機構口罩剩餘數量明細清單 - GraphQL/JSON 開放版](https://mask-data-farmer.herokuapp.com/)

# Techtechnology

- nodejs
- graphQL
- cronJob

# Installation

```
> git clone https://github.com/madeinfree/mask-data-farmer.git
> cd mask-data-farmer
> yarn
```

# Uses

```
> npm start
> open http://localhost:53310
```
